Spacewalk project has been discontinued on May 31 2020.

  • Spacewalk is installable on Fedora 30 and 31
  • Spacewalk supports Fedora 30 and 31, Red Hat Enterprise Linux and CentOS 8 clients
  • Number of bugfixes and security fixes

  • Spacewalk nightly contains an experimental code for RHEL8 server (with dnf backend which breaks RHEL / CentOS 6 support).

The up-to-date API documentation can be found at http://spacewalkproject.github.io/documentation/api/2.10/
