Latest Source

The latest source code is available from the spacewalkproject/spacewalk repository on GitHub

You can clone a copy with

git clone https://github.com/spacewalkproject/spacewalk.git

For more information please visit the Spacewalk developer wiki.

Source RPMs

Source RPMs can be found at the same location as our binaries. Older source RPMs (pre Spacewalk 1.8) can be found here.

Binary RPMs

RPM binaries for the project are available for download via a yum repository here. Our RPM packages are GPG signed. Visit the download area of the project wiki for more information and access to the Spacewalk public GPG key. For more information on using this yum repository and installing Spacewalk, please visit the Spacewalk installation procedures on the developer wiki.
